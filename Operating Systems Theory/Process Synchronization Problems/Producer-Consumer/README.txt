To run the code:

1. Open Windows command prompt
2. Change your directory to the location you have placed this folder ('project4')
3. Ensure you have downloaded a Java Developers Kit (JDK)
	I downloaded mine from Oracle's website: https://www.oracle.com/java/technologies/downloads/#jdk21-windows
4. Compile the code with this command: 
	javac ProducerConsumer.java
5. Run the code with this command:
	java ProducerConsumer
